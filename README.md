## 📌 Deskripsi Proyek

**E-Commerce Ludonanzak** adalah sebuah website toko online yang dibuat sebagai tugas akhir mata pelajaran Pemrograman Web. Website ini menggunakan tema coklat-orange (**Cozy Theme**) dan menampilkan berbagai kategori produk yang dapat dilihat dan dipilih oleh pengguna, dibangun menggunakan teknologi web dasar seperti HTML, CSS, dan JavaScript — tanpa framework atau library eksternal.

## ✨ Fitur Utama

### 🛍️ Belanja
- **Keranjang Belanja** — Tambah, hapus, dan kelola produk sebelum checkout
- **Diskon Produk** — Tampilan harga coret dan persentase diskon pada produk tertentu

### 🔍 Pencarian & Navigasi
- **Pencarian Produk** — Cari produk dengan cepat menggunakan search bar
- **Sort / Urutkan Item** — Urutkan produk berdasarkan harga, nama, atau lainnya

### 📦 Produk
- **Deskripsi Produk** — Penjelasan singkat produk ditampilkan pada setiap item

### 🎨 Tampilan
- **Dark Mode / Light Mode** — Tema terang dan gelap yang bisa diubah sesuai preferensi pengguna
- **Responsive Design** — Tampilan menyesuaikan berbagai ukuran layar (mobile, tablet, desktop)
- **Cozy Theme** — Palet warna coklat-orange yang hangat dan nyaman di mata

## 🚀 Cara Menjalankan

### ✅ Cara 1 — Clone & Jalankan Lokal

1. Clone repository ini:
   ```bash
   git clone https://github.com/luckqfz/E-Commerce
   ```

2. Masuk ke folder proyek:
   ```bash
   cd E-Commerce
   ```

3. Buka file `index.html` di browser favorit kamu, atau gunakan ekstensi **Live Server** di VS Code untuk auto-reload saat ada perubahan.

### 🌐 Cara 2 — Akses Langsung Online

1. Buka browser kamu
2. Kunjungi **[luckqfz.github.io/E-Commerce](https://luckqfz.github.io/E-Commerce)**
